- 视角名称: 心理学视角
- 核心问题: 行为背后的心理机制是什么？
- 适用场景: 需要深度理解行为动机、分析关系模式、探究心理成因的问题
- 理论基础: 心理学各分支学科（发展心理学、社会心理学、进化心理学、积极心理学等）
- 视角分类: 学科视角、认知性视角/结构性视角、解释型视角/转换型视角、主视角、从学科创造、通用视角
- 适用对象: 人物型/现象型/系统型、动态/历史、复杂/混沌、理解型/解释型/转换型、深层/根源、单一/关联/嵌套
- Root Rank形态: 反馈循环

## 核心定义层
### 什么是心理学视角
心理学视角不是贴标签（性格、人品、意志力），不是给建议（你应该怎样做），而是看见行为背后的生成器——那些看不见的心理机制、认知模式、情绪回路、社会动力，才是真正驱动行为的引擎。

### 核心概念
**生成器**: 行为不是自发的，而是由特定心理机制生成的。理解生成器，才能从源头上改变。
**循环因果**: 真实的关系往往是相互塑造的，不是单向的A导致B。你的行为影响他人，他人的反应又反过来影响你。
**多元真相**: 同一现象在不同视角下可以有完全不同的"真相"，这不是相对主义，而是承认复杂系统的多维度。
**进化设定**: 我们的心理系统不是为了让我们"幸福"，而是为了让我们"生存"。理解这个设定，才能不被情绪牵着走。
**依恋模式**: 早期经历塑造了我们的关系模式，看见这个模式，才能在亲密关系中不重复旧剧本。

### Root Rank形态
心理学视角的root rank形态为反馈循环，其关系本质是心理机制与行为相互塑造，形成正向或负向反馈回路，适合用反馈环路图(标 +/-)来可视化。

### 核心创新
心理学视角采用"要点索引+问题匹配"机制，只输出问题最相关的1-2个核心要点，深度展开分析，避免泛泛而谈。

## 要点索引层

### 要点1: 看见行为背后的生成器

**核心判据**:
- 是否需要理解行为背后的心理机制而非表象判断？
- 是否涉及情绪、认知、动机等不可见的心理过程？
- 是否需要从"为什么"层面解释行为？

**适用场景**: 需要深度理解行为动机、分析极端行为、探究心理成因的问题

**典型案例**: 米勒分析纳粹高官早年经历与有毒教育的关系；从犯罪心理学视角分析多重人格与犯罪的关联

### 要点2: 从线性因果到循环因果

**核心判据**:
- 问题是否涉及多个人或系统的相互影响？
- 是否需要打破"受害者"心态，看见自身在循环中的作用？
- 是否涉及关系模式、互动模式的分析？

**适用场景**: 关系冲突、沟通问题、团队动力、家庭系统等问题

**典型案例**: 循环因果视角vs线性因果视角在"第三选择"中的应用；榜一大哥现象中的软成瘾分析

### 要点3: 多元真相与边界感

**核心判据**:
- 是否涉及不同立场的冲突或误解？
- 是否需要容纳对立的情绪和观点？
- 是否需要从单一真相转向多元共存？

**适用场景**: 沟通冲突、观点分歧、需要理解不同立场的问题

**典型案例**: 心理学视角下的多元真相，强调边界感和容纳情绪共存；手足竞争理论中孩子对"爱"的感知差异

### 要点4: 进化设定的心理机制

**核心判据**:
- 是否涉及情绪、欲望、满足感等基础心理体验？
- 是否需要理解"为什么人类会这样"的进化逻辑？
- 是否涉及短期快乐vs长期价值的冲突？

**适用场景**: 情绪管理、动机分析、成瘾行为、幸福感问题

**典型案例**: 进化心理学视角下的佛学——快乐短暂是进化的设定；软成瘾机制：对短期确定快乐的不加节制渴求

### 要点5: 依恋模式与早期创伤

**核心判据**:
- 是否涉及亲密关系、人际关系模式？
- 是否需要理解当前行为与早期经历的关联？
- 是否涉及安全感、信任、边界等问题？

**适用场景**: 亲密关系问题、人际障碍、自我价值感问题

**典型案例**: 1900的孤独感：存在主义焦虑与矛盾型依恋的交织；安全依附理论在手足竞争中的应用

## 匹配逻辑层

### 问题特征分析维度

**问题类型**: 决策型/解释型/预测型/转换型

**关键要素**: 不确定性/新信息/多选项/历史数据/持续调整

**问题尺度**: 个人层面/组织层面/系统层面

### 匹配度计算公式

匹配度 = (类型匹配度 × 0.4) + (要素匹配度 × 0.4) + (尺度匹配度 × 0.2)

### 输出规则

- 只输出匹配度最高的1-2个点
- 如果最高匹配度<0.5，说明心理学视角不适用

## 操作工序层

### 第一步: 问题特征分析与要点匹配

**说明**:  
这是心理学视角的核心创新，不是全量输出所有要点，而是先分析问题特征，匹配最相关的1-2个点，然后只围绕这些点深度展开。

**方法**:

- 分析问题类型: 决策型/解释型/预测型/转换型
- 识别关键要素: 不确定性/新信息/多选项/历史数据/持续调整
- 确定问题尺度: 个人层面/组织层面/系统层面
- 计算每个要点的匹配度
- 选出匹配度最高的1-2个点
- 如果最高匹配度<0.5，说明心理学视角不适用，返回"不适用"判断

**输出格式**:

```
## 问题特征分析与要点匹配

### 问题特征分析

**问题类型**: [决策型/解释型/预测型/转换型]

**关键要素**: [列出问题包含的关键要素]

**问题尺度**: [个人层面/组织层面/系统层面]

### 要点匹配结果

**选中要点1**: [要点名称] (匹配度: [0.XX])

**选中要点2**: [要点名称] (匹配度: [0.XX])

### 匹配度说明

[简要说明为什么选中这些点]
```

**后续步骤**: 围绕选中的要点深度展开分析，每个要点详细说明核心原理、用判据过一遍问题、给出具体分析结论。

### 第二步: 基于选中要点深度分析

**说明**:  
围绕选中的1-2个要点，详细说明核心原理，用判据过一遍问题，给出基于心理学的具体分析结论。

**方法**:

- 说明选中要点的核心原理（基于心理学理论）
- 用要点的判据逐条分析问题
- 给出具体的分析结论和洞察
- 如果涉及多个要点，说明要点之间的关联

**输出格式**:

```
## 深度分析

### [要点名称]的分析

**核心原理**: [说明这个要点的心理学原理]

**判据分析**:
- 判据1: [分析]
- 判据2: [分析]
- 判据3: [分析]

**分析结论**: [基于心理学的具体结论]

### [要点名称]的分析

[重复上述结构]
```

### 第三步: 给出实践建议

**说明**:  
基于心理学分析，给出可操作的实践建议，帮助用户将洞察转化为行动。

**方法**:

- 基于分析结论，提炼2-3个关键建议
- 每个建议具体、可执行
- 说明建议背后的心理学逻辑

**输出格式**:

```
## 实践建议

**建议1**: [具体建议]
- 心理学逻辑: [说明]

**建议2**: [具体建议]
- 心理学逻辑: [说明]
```

## 判据层

在开始分析前，先过一遍这四条判据，确保你的分析是心理学视角的:

**判据1**: 是否看见了行为背后的心理机制，而非停留在表象判断？
**判据2**: 是否从循环因果、多元真相、进化设定等心理学核心概念出发？
**判据3**: 是否基于心理学理论（而非常识或个人经验）给出分析？
**判据4**: 是否给出了可操作的实践建议，帮助用户将洞察转化为行动？

## 结构判断层

### 双闸判断

**闸1: 心理机制可见性**

- 问题是否涉及不可见的心理过程（情绪、认知、动机）？
- 是否需要从"为什么"层面解释行为？
- 是否有心理学理论可以支撑分析？

**闸2: 生成器可识别性**

- 是否能识别出行为背后的心理生成器？
- 是否能找到心理机制与行为之间的反馈回路？
- 是否能从心理学视角给出独特洞察？

**判断逻辑**:

- 心理机制可见性高 + 生成器可识别性高 = 心理学视角高度适用
- 心理机制可见性高 + 生成器可识别性低 = 心理学视角中度适用
- 心理机制可见性低 + 生成器可识别性高 = 心理学视角低度适用
- 心理机制可见性低 + 生成器可识别性低 = 心理学视角不适用

## 反坍缩闸

### 避免常见陷阱

**陷阱1: 贴标签而非看见生成器**

- 症状: 用"性格""人品""意志力"等标签解释行为，没有深入心理机制
- 对策: 始终追问"为什么"，直到看见行为背后的心理生成器

**陷阱2: 线性因果而非循环因果**

- 症状: 用单向因果解释问题，忽视相互塑造的反馈回路
- 对策: 从循环因果视角出发，看见系统中的相互影响

**陷阱3: 单一真相而非多元真相**

- 症状: 认为只有一个正确解释，排斥其他视角
- 对策: 承认真相的多元性，容纳对立的情绪和观点

**陷阱4: 缺乏心理学理论支撑**

- 症状: 基于常识或个人经验给出分析，没有引用心理学理论
- 对策: 基于心理学理论（发展心理学、社会心理学、进化心理学等）给出分析

**陷阱5: 分析结论无法落地**

- 症状: 只有理论分析，没有可操作的实践建议
- 对策: 基于分析结论，给出具体、可执行的建议

**陷阱6: 匹配失败强行输出**

- 症状: 所有要点的匹配度都<0.5，但仍强行输出分析
- 对策: 明确返回"心理学视角不适用"，并说明原因

## 写作规范层

### 输出结构

1. 问题特征分析与要点匹配
2. 深度分析（基于选中的要点）
3. 实践建议

### 写作风格

- 零AI腔: 禁止"根据数据显示、系统分析表明、深入探讨、至关重要、此外、进一步、值得注意的是"
- 零咨询师腔: 禁止"这恰恰说明、这正是、这其实反映了"
- 零套话: 禁止"希望对你有帮助、加油、继续努力、坚持就是胜利"
- 零泛夸: 禁止"很棒、很好、很有想法、很有深度、不错"
- 口语化: 用"你"不用"您"，说人话，像跟聪明朋友聊天
- 短句优先: 能用两个字说的不用四个字
- 一句一事: 每句只推进一步，长句拆短
- 具体: 名词看得见，动词有力气

### 格式要求

- 加粗标题用 **XX** 格式，每个标题后必须空一行
- 不用 markdown 引用块
- 不用「」括号

## ASCII结构图

**Root Rank形态**: 反馈循环  
**取景框**: 反馈环路图(标 +/-)

```
     +-------------------+
     |   心理机制        |
     | (情绪/认知/动机)  |
     +---------+---------+
               |
               | (+)
               v
     +-------------------+
     |      行为         |
     +---------+---------+
               |
               | (+/-)
               v
     +-------------------+
     |   环境反馈        |
     | (他人反应/结果)  |
     +---------+---------+
               |
               | (+)
               v
     +-------------------+
     |   心理机制        |
     | (强化/调整/修正)  |
     +-------------------+

示例: 软成瘾的反馈循环

     +-------------------+
     |   渴求快乐        |
     | (短期确定性刺激)  |
     +---------+---------+
               |
               | (+)
               v
     +-------------------+
     |   打赏行为        |
     | (榜一大哥)       |
     +---------+---------+
               |
               | (+) 获得即时满足
               v
     +-------------------+
     |   强化渴求        |
     | (多巴胺奖励)     |
     +---------+---------+
               |
               | (+)
               v
     +-------------------+
     |   渴求增强        |
     | (耐受性上升)     |
     +-------------------+
```

**说明**: 这个图展示了心理学视角的核心——心理机制与行为之间的反馈循环。正向循环(+)会强化原有模式，负向循环(-)会抑制或调整行为。看见这个循环，才能从源头上改变模式。
